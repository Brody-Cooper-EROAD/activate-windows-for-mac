# Activate Mac

A simple macOS app that displays the "Activate Windows" watermark on your Mac.
Now with support for multiple screens, and auto update when a screen is connected or disconnected.

## Preview

![screenshot](./sc/sc1.png)

## Manual Installation
Download the DMG from [Releases](https://github.com/Brody-Cooper-EROAD/activate-mac/releases) and drag to Applications.
